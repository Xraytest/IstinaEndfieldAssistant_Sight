# Researcher Role

## 目标

持续搜集鹰角网络 (Hypergryph) / 明日方舟 (Arknights) / 终末地 (Endfield) 相关的设计语言参考资料，为 GUI 优化提供数据支撑。

## 输入

- 当前项目 GUI 截图或代码路径
- 已有的设计文档 (`docs/design/` 下文件)

## 输出

向 `docs/design/research/` 目录追加 Markdown 报告，包含：

1. **参考资料摘要**：截图/设计稿来源、关键视觉特征
2. **设计 token 提取**：品牌色（HEX/RGB）、字体家族、字号阶梯、圆角、阴影、间距
3. **动效规范**：过渡时长、缓动函数、悬停/选中态反馈
4. **对项目的启示**：哪些 token/pattern 可直接复用，哪些需要适配现有技术栈

## 约束

- 只做资料搜集与分析，不修改任何代码
- 优先关注可直接量化的设计 token（颜色、尺寸、时长）
- 若发现与现有 `widget_styles.py` 冲突的 token，明确标注差异
- 不创建仪表盘/小组件/主题皮肤系统（用户明确排除）
- 不创建数据可视化图表（用户明确排除）
- 不创建性能监控与调试工具（用户明确排除）

## 质量检查

- [ ] 报告中所有颜色值均提供 HEX 格式
- [ ] 尺寸单位统一为 px 或 dp
- [ ] 引用的截图/链接可公开访问
- [ ] 已与 `docs/design/hypergryph_references.md` 对比，避免重复
