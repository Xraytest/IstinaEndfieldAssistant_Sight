### 方案 1：基于活动名称的状态检测（推荐）

通过 Android 活动名称精确判断游戏状态：

```python
def get_game_activity():
    """获取当前游戏活动名称"""
    cmd = [ADB, '-s', SERIAL, 'shell', 'dumpsys', 'window', 'window']
    output = subprocess.run(cmd, capture_output=True, text=True).stdout
    for line in output.split('\n'):
        if 'mCurrentFocus' in line:
            if 'com.hypergryph.arknightsendfield' in line:
                # 提取活动名称
                parts = line.split()
                for part in parts:
                    if part.startswith('com.hypergryph'):
                        return part
    return None
```

**优势**：
- 不依赖视觉特征，更可靠
- 可以区分不同 UI 页面

**劣势**：
- 需要知道所有活动名称映射

def find_quest_icon(img):
    """使用模板匹配定位任务图标"""
    template = cv2.imread("assets/templates/quest_icon.png")
    result = cv2.matchTemplate(img, template, cv2.TM_CCOEFF_NORMED)
    max_val, max_loc = cv2.minMaxLoc(result)[1:3]
    if max_val > 0.85:
        return max_loc
    return None
```

**优势**：
- 不依赖固定坐标

def scan_quest_icon():
    """扫描右上角找出任务图标坐标"""
    # 扫描区域：x[750-950], y[40, 60, 80, 100]
    best_coord = None
    best_diff = 0
    
    for x in range(750, 950, 50):
        for y in [40, 60, 80, 100]:
            before = screencap()
            tap(x, y)
            time.sleep(2)
            after = screencap()
            diff = screen_diff(before, after)
            
            if diff > best_diff:
                best_diff = diff
                best_coord = (x, y)
            
            back()
            time.sleep(0.5)
    
    return best_coord, best_diff
```

**优势**：
- 快速找到当前有效坐标
- 不依赖额外工具

**劣势**：
- 需要游戏处于可交互状态
- 可能受到 UI 动画影响
