# IEA 宣传视频核心概念

## 项目名称
**IstinaEndfieldAssistant (IEA)**

## 核心主题
**完全智能 · 解放双手**

## 目标受众
- 《明日方舟：终末地》玩家
- 寻求自动化解决方案的游戏玩家
- 对 AI 自动化技术感兴趣的技术爱好者

## 视频定位
- **时长**: 60 秒
- **风格**: 极简科技感，参考 Hypergryph 视觉语言
- **色调**: 深蓝/青色主调 (#18D1FF, #00FFA2)，黑色背景 (#0A0A0F)
- **节奏**: 快慢交替，关键信息停顿强调
- **原则**: 无花哨动画，聚焦核心功能

## 品牌关键词
1. **智能** - VLM 视觉理解，自主决策
2. **解放** - 自动化执行，无需手动操作
3. **安全** - 本地/云端双模式，加密通信
4. **专业** - 终端风格 UI，技术感强烈

## 参考风格
- Hypergryph 官方宣传片：几何线条、数据流动、科技感 UI
- 避免：过度粒子效果、花哨转场、卡通风格
