## 新增发现（1 项，Medium）

### OO-03 — prts_full_intelligence_page.py:231 QLineEdit.text 调用错误

**文件**: `src/gui/pyqt6/pages/prts_full_intelligence_page.py:231`
**代码**:
```python
text = self._prompt_input.text.strip()
```
**问题**: `QLineEdit.text` 在 PyQt6 中是方法（需 `text()` 调用），不是 Python 属性。`self._prompt_input.text` 返回 bound method 对象，对其调用 `.strip()` 会抛出 `AttributeError: 'builtin_function_or_method' object has no attribute 'strip'`，导致用户在 PRTS 页面发送聊天消息时直接崩溃。
**严重度**: Medium（PRTS 页面标记"施工中"，用户尚未大规模使用，但代码路径已激活）
**建议**: 改为 `text = self._prompt_input.text().strip()`
**状态**: 新增 Open 条目

**证据**: 全仓库仅此一处使用 `.text.strip()` 未加括号。其余所有 QLineEdit 使用均正确调用 `.text()`（如 recorder.py:167、settings_page.py 等）。
