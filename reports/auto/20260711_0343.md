### 1.1 W1 (Critical) — VLM 行走导航字母键被 `_is_valid_keyevent` 拒绝

**当前状态：仍存在（未修复）**

`android_runtime.py:75-84`：
```python
def _is_valid_keyevent(key: Any) -> bool:
    if key is None:
        return False
    s = str(key).strip()
    if not s:
        return False
    if s.isdigit():
        return True
    return s in _KNOWN_KEYEVENT_NAMES
```

白名单仍仅接受纯数字和 `_KNOWN_KEYEVENT_NAMES` 中的常量名。`vlm_walk_navigator.py` 的 `_execute_action` 仍发送字母键 `"w"/"a"/"s"/"d"/"q"/"e"/"f"`。整条 VLM 行走导航链路（`nav3 walk` / `nav3 to_entity`）**仍完全失效**。

**当前状态：仍存在（未修复）**

`theme_manager.py:393-396`：
```python
def __new__(cls) -> "ThemeManager":
```
